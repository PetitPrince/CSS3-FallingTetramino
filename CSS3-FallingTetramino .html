CSS3 Falling Tetromino Effect

Hello !

This page is a demo to show how to get a falling Tetromino background, purely in CSS3. It was originally conceived while making the custom CSS file for my blog dedicated to Tetris at Wefraga>. Wefrag uses a pretty old version of Wordpress Mu, so I had to work with some constraint while designing the blog. Fortunately, the default theme have pretty designer-friendly layout and more importantly, some empty div freely usable.

The effect make full use of those extra div. I styled each of them individually and then used the :after pseudo-element in order to have more thing to play with. The animation is done by the means of the @keyframe statement.

If you want to learn more, look at the source of this page !
